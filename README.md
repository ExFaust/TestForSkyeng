# TestForSkyeng

Описание:

   Приложение отображает все возможные переводы, связанные с введённым словом, а также отображает дополнительный экран с подробной информацией о переводе. 
   Второй вариант исполнения, который вероятно требовался, отображать для выбора сначала связанные с введённым слова, выбирать одно и только потом показывать переводы для него.   (Переделать не трудно) 

Инструкция по установке:

   В корне репозитория лежит файл .apk, который нужно сбросить на любой девайс с апи 16+ и установить.

Описание используемых библиотек:

    glide - загрузка картинок
    dagger2 - DI
    retrofit2/okhttp3 - запросы в сеть
    rxjava2 - работа с потоками
    остальные входят в пакет androidx
